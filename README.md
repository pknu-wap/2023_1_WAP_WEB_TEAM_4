GLOG는 사용자가 게시물을 작성하여 업로드하고, 다른 사용자들과 게시물을 공유하며 소통하는 블로그 웹 서비스 입니다.

# 주요기능
1. 게시물 검색 기능
2. 사용자 검색 기능
3. 좋아요순 게시물 조희
4. 조회수순 게시물 조회
5. 랜덤 게시물 조회
6. 다크모드와 라이트모드
